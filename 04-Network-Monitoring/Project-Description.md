# Network Monitoring Project

## Objective
Monitor network performance and detect connectivity issues.

---

## Tools Used
- SolarWinds

---

## Implementation

- Added network devices (switches, routers)
- Configured SNMP monitoring
- Set bandwidth utilization alerts
- Created performance dashboard

---

## Incident Example

Detected abnormal bandwidth spike:
- Identified source device
- Verified running services
- Determined misconfigured backup job
- Adjusted schedule
