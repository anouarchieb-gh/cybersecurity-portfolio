# Active Directory Security Hardening

## Objective
Strengthen domain security using best practices and policy enforcement.

---

## Environment
- Windows Server 2019
- Active Directory Domain Services
- Group Policy Management

---

## Security Improvements

- Enforced password complexity
- Configured account lockout threshold
- Restricted Domain Admin privileges
- Disabled unused accounts
- Structured OU separation

---

## GPO Hardening

- Disabled anonymous SID enumeration
- Restricted local administrator usage
- Enforced screen lock timeout
- Enabled auditing policies

---

## Monitoring

- Monitored Event ID 4720 (user creation)
- Monitored privilege escalation attempts
- Audited failed logon events

---

## Skills Demonstrated

- Identity & Access Management
- AD Security
- GPO Hardening
- Security Policy Enforcement
